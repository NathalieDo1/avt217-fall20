Instructions

1. Open XD

2. Select Custom Size/Artboard

3. Select the square-shape tool on the left handside

4. Create a brown (#653416), 560 width x 408 height recatangle shape, with no border

5. Place the rectangle at x: -1032 and y: 1694

6. Next, select the triangle-shape tool from the left

7. Create an olive green (#4A5A24), 1595 width x 1381 height triangle shape, with no border

8. Place the triangle at x: -1550 and y: 313

9. Select the triangle-shape tool from the left again

10. Create another lighter olive green (#516425), that is 1405 width x 1216 height triangle, with no border

11. Place that triangle at x: -1455 and y: -156

12. Create another lighter olive green (#617434), that is 1055 width x 913 height triangle, with no border

13. Place that triangle at x: -1280 and y: -493

14. Lastly, select the circle-shape tool on the left

15. Create a yellow circle (#E5BB15), 307 width x 307 height, with no border

16. Place the circle at x: -905 and y: -646

17. Select the 3 lines at the top left, export your file as a PNG

18. Email your results to ndo21@masonlive.gmu.edu
